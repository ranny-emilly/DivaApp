# 

Sistema de gerenciamento de uma loja de roupas femininas desenvolvido em Java para a disciplina de Programação Orientada a Objetos.

##  Objetivo

Desenvolver uma aplicação desktop aplicando os principais conceitos de Programação Orientada a Objetos, incluindo encapsulamento, herança, polimorfismo, abstração, interfaces, persistência em arquivos e tratamento de exceções.
